# Kernel-of-Linux
리눅스 커널에 대해 공부하고 정리하는 레포지토리입니다.


# 계획
1. **운영체제 기본 개념**
	- 프로세스, 스레드, 스케줄링
	- 메모리 관리 기초 (가상 메모리, 페이징, 세그먼테이션)
	- 파일 시스템 개요 및 구조
	- 프로세스 간 통신 (IPC) 기본 개념
	- 디바이스 드라이버 소개
	  
2. **리눅스 시스템 사용**
	- 기본 명령어
	- 파일 시스템 계층 구조
	- 패키지 관리 및 사용자 권한 관리
	- 쉘 스크립팅
	  
3. **시스템 프로그래밍**
	- 시스템 콜 개념 및 사용법
	- 시그널 (Signal) 처리
	- 메모리 매핑 (mmap) 기법
	- POSIX API 활용
	  
4. **커널 핵심 서브시스템**
	-  프로세스 스케줄러: CFS 동작 원리
	- 메모리 관리 유닛 (MMU): 가상 메모리, 페이지 테이블, TLB
	- 가상 파일 시스템 (VFS): 다양한 파일 시스템 인터페이스 통합
	- 네트워킹 서브시스템: TCP/IP 스택, 소켓 인터페이스
	- I/O 시스템 및 블록 레이어
	  
5. **커널 구조 이해**
	- 모놀로식 커널 vs 마이크로커널 비교
	- 커널 스페이스 vs 유저 스페이스 분리
	- 보호 링 구조: Ring 0과 Ring 3
	- 메모리 존: DMA, NORMAL, HIGHMEM
